# SIEM 

- Since log sources generate numerous logs as there are various types of challenges associated with it we use SIEM

- SIEM = Security Information and Event Management System 

-  is a security solution that 
	- collects logs from various types of log sources
	- standardizes their format into a consistent one
	- correlates them
	- detects malicious activities using detection rules

- features of SIEM:
	- centralized log collection: logs from all places(endpoint, firewall, server..) are centralized in one place
	- normalization of logs: all logs are in the same format 
	- correlation of logs: finds relationship between different logs 
	- real time alerting: detection rules can be made and alerts can be triggered to notify analyst
	- Dashboards and Reporting: the summary of the analysis can be in the form of analysis
